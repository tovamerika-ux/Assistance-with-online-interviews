# Помощь в онлайн-собеседовании

Документация по настройке аудио, использованию ИИ-ассистентов и человека-помощника на онлайн-собеседованиях.

## Документы

1. **[Звонки с телефона через ПК](md-docs/звонки-через-пк.md)**
   Как передавать телефонный разговор на ПК для работы с нейронкой, сохраняя качество звука для собеседника.

2. **[Схемы аудио потоков с одной USB-гарнитурой](md-docs/аудио-потоки-usb-headset.md)**
   Использования невидимых ассистентов-нейронок на ПК во время онлайн-собеседований. Усиление звука, подключение человека-помощника через Discord, подавление шума — всё на базе Voicemeeter и одной USB-гарнитуры.

3. **[Незаметный человек-помощник, TWS-наушники](md-docs/человек-помощник-tws.md)**
   Схемы подключения человека-помощника через TWS-наушники и использование ИИ-ассистентов (Shadowhint, Sobes Copilot).

4. **[Восстановление уровня громкости и шумодав](md-docs/восстановление-громкости.md)**
   Решение проблем с заниженной громкостью микрофона и тихим звуком от собеседника при помощи Voicemeeter, шумодав через Krisp.

## Используемое ПО

- [Voicemeeter](https://vb-audio.com/Voicemeeter/) — виртуальный аудио-микшер
- [Krisp](https://krisp.ai/) — шумоподавление
- [SoundVolumeView](https://www.nirsoft.net/utils/sound_volume_view.html) — управление громкостью из командной строки
