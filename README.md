# Помощь в онлайн-собеседовании

Документация по настройке аудио, использованию ИИ-ассистентов и человека-помощника на онлайн-собеседованиях. Для Windows 10/11 пользователей.

Сборник практических инструкций по настройке звука и «невидимых помощников» для **онлайн-собеседований** и созвонов на **Windows 10/11** — всё на базе бесплатного виртуального микшера **Voicemeeter** (Standard / Banana / Potato) и шумодава **Krisp**. Разобрано, как **передать телефонный разговор на ПК в прямом эфире** через «Связь с телефоном» (**Phone Link**) в обход блокировки записи звонка на Android (Samsung), как **усилить тихий голос собеседника и свой микрофон** в **Teams / Zoom / Яндекс Телемост / Discord**, как **подавить шум клавиатуры и мыши**, отдать «чистый» звук невидимым ИИ-ассистентам **ShadowHint** и **Sobes Copilot** и захватить системный звук в **OBS** (в том числе через **WASAPI loopback** и виртуальные шины Voicemeeter).

Отдельно рассмотрены **схемы аудио-потоков для одной USB-гарнитуры**, **незаметное подключение человека-помощника через Discord и TWS-наушники** (BT-адаптеры TX/RX, режим A2DP, ответы в Телеграм), а также частые проблемы: **OBS не пишет системный звук с Voicemeeter** (настройка **Enable Loopback Streaming**), «сползание» и занижение громкости микрофона при 100%, слабое усиление на **Realtek-чипе**, тишина в гарнитуре, конфликт **двух BT-гарнитур в Windows**, разница **pre-Krisp / post-Krisp**. По сути — практический гайд по «**невидимому ассистенту на собеседовании**» и настройке звука для **interview copilot**.

## Используемое ПО

- [Voicemeeter](https://vb-audio.com/Voicemeeter/) — виртуальный аудио-микшер
- [Krisp](https://krisp.ai/) — шумоподавление
- [SoundVolumeView](https://www.nirsoft.net/utils/sound_volume_view.html) — управление громкостью из командной строки
