# Помощь в онлайн-собеседовании

Документация по настройке аудио, использованию ИИ-ассистентов и человека-помощника на онлайн-собеседованиях.

## Документы

1. **[Звонки с телефона через ПК](md-docs/звонки-через-пк.md)** (v4.1)
   Как передавать телефонный разговор на ПК для работы с нейронкой, сохраняя качество звука для собеседника.

2. **[Схемы аудио потоков с одной USB-гарнитурой](md-docs/аудио-потоки-usb-headset.md)** (v2.9)
   Использования невидимых ассистентов-нейронок на ПК во время онлайн-собеседований. Усиление звука, подключение человека-помощника через Discord, подавление шума — всё на базе Voicemeeter и одной USB-гарнитуры.

3. **[Незаметный человек-помощник, TWS-наушники](md-docs/человек-помощник-tws.md)** (v2.7)
   Схемы подключения человека-помощника через TWS-наушники и использование ИИ-ассистентов (Shadowhint, Sobes Copilot).

4. **[Восстановление уровня громкости и шумодав](md-docs/восстановление-громкости.md)** (v3.1)
   Решение проблем с заниженной громкостью вашего микрофона и тихим звуком от собеседника при помощи Voicemeeter, шумодав через Krisp.

Оригинальные Word-документы: **[DOCX/](DOCX/)**

> **На заметку:** тут десятки страниц текста, скриншотов и кода, экономьте время, не читайте сами — запустите сюда нейронку (Claude Code, Codex), или скидывайте в обычный чат нейронки доку (docx) и просите её собрать нужную вам аудио конфигурацию. И она это сделает уже с минимум ошибок и фантазий, т.к. в доках есть самое ценное — прописаны ограничения всех элементов системы, из которых собирается аудио граф.

## Используемое ПО

- [Voicemeeter](https://vb-audio.com/Voicemeeter/) — виртуальный аудио-микшер
- [Krisp](https://krisp.ai/) — шумоподавление
- [SoundVolumeView](https://www.nirsoft.net/utils/sound_volume_view.html) — управление громкостью из командной строки
