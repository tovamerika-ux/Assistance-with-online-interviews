Это папка со схемами для WORD-инструкций.

PlaintUML-код для диаграмм создан при помощи ChatGPT 5.2 Plus режим Auto.
Отрисовка тут:
https://www.plantuml.com/plantuml/uml/

Номера 3.1, 3.2,.. 4.1,4.2 в начале имени файла - это номер раздела в соответствующей Word-инструкции.


===================
Ниже промт для ChatGPT
ПРАВИЛА PLANTUML (чеклист, чтобы не ловить ошибки)

1) НЕ писать цепочки стрелок в одну строку
   ❌ A --> B --> C
   ✅ A --> B
      B --> C

2) В rectangle / component:
   - физический перенос строки ЗАПРЕЩЁН
   - перенос строки ТОЛЬКО через \n
   ❌ rectangle "Line1
      Line2"
   ✅ rectangle "Line1\nLine2"

3) В note:
   - \n НЕ использовать
   - ТОЛЬКО физические переносы строк
   ❌ note right of X : line1\nline2
   ✅
      note right of X
      line1
      line2
      end note

4) Если note right of X рендерится пустым или криво,
   использовать стабильную форму:
   note as NOTE1
   text...
   end note
   NOTE1 .. X

5) Для максимальной совместимости с plantuml.com:
   - использовать rectangle
   - избегать componentStyle rectangle, если появляются пустые блоки
   - избегать сложных skinparam

6) Если появляется ошибка:
   "Syntax Error? (Assumed diagram type: activity / component)"
   проверить в первую очередь:
   - нет ли физического переноса строки внутри "..."
   - нет ли цепочки стрелок
   - не сломан ли note

7) В лейблах rectangle:
   - избегать спецсимволов
   - избегать слишком длинных строк
   - при необходимости дробить через \n
